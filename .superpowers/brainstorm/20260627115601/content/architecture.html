Transport Layer: UDP → QUIC

Current:  saferunnet-transport/
          ├── udp.rs       (raw UDP socket)
          └── transport.rs (bare wrapper)

Target:  saferunnet-transport/
         ├── quic.rs         (quinn::Connection wrapper)
         ├── datagrams.rs    (QUIC unreliable datagram per RFC 9221)
         ├── control.rs      (BTRequestStream → bidirectional stream)
         ├── link.rs         (LinkManager — merged from saferunnet-link)
         ├── event.rs        (Event loop abstraction)
         ├── tcp_tunnel.rs   (TCPHandle / TCPConnection)
         └── traits.rs       (TransportLayer trait for testing)

Observability: 24-line → Full spdlog-level

saferunnet-observability/
├── sink.rs        (Print / File / System syslog sinks)
├── category.rs    (per-module log level: router=debug, crypto=warn...)
├── ringbuf.rs     (ring buffer for RPC log subscription)
├── callback.rs    (callback sink for external consumers)
├── config.rs      (LoggingConfig from config file)
└── init.rs        (init_logging from config)
