## What Did Not Change

- no network transport, socket IO, or link protocol implementation yet
- router protocol coverage is still limited to one announcement family
- link protocol coverage is still limited to one ping-style control variant plus the initial typed session-init, session-accept, and session-path-switch families
- no typed service graph or dependency policy enforcement yet
- the persisted identity format is still simple text and not yet Lokinet-compatible
- project-owned secret-key APIs still expose explicit copy-based export methods for serialization and backend interop
- no DH encryption, transport, or persistence exists for the current session lifecycle slice yet
- app runtime now publishes shared session state, but orchestration is still minimal and entirely in-memory
